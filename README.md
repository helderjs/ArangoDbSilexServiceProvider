Provides ArangoDB API as services to Silex applications.

Features
--------

* Multiple Databases connections
* Document/Collection Management
* Graph/Edge Management
* AQL Statement

Requirements
------------

* PHP 5.6+
* ArangoDB-PHP (Tested with 3.1)
* Silex 1.3
